🧰 IT Engineer / Manager Tool Stack (2026)
==========================================

🤖 AI & LLM Tools (Free / OSS-friendly focus)
---------------------------------------------

### General AI / Thinking

*   **[ChatGPT](https://chat.openai.com/)** / **[Claude](https://claude.ai/)** (free tiers) – reasoning, docs, debugging, design reviews
    
*   **[Ollama](https://ollama.com/)** – private, offline AI (Llama, Mistral, DeepSeek)
    

### IDE / Vibe Coding

*   **[VS Code](https://code.visualstudio.com/)**  
*   **[Continue.dev (OSS)](https://continue.dev/)** – repo-aware AI editing  
*   **[Codeium (free)](https://www.codeium.com/)** – inline completion + chat  
*   **Cursor-style workflows** (multi-file AI edits)  
*   **[OpenAI Codex](https://openai.com/codex/)** – structured repo transforms, refactors, test generation  
*   **[Claude Code](https://claude.ai/code)** – agentic multi-file edits, interactive coding  

### Terminal-First AI (Power-user tier)

*   **[antigravity](https://antigravity.google/)** – natural language → shell, explain/fix commands  
*   **[llm (Simon Willison)](https://llm.datasette.io/)** – pipe text/logs/diffs into LLMs  
*   **[aider](https://aider.chat/)** – AI pair programming directly in git repos  
*   **[shell-gpt (sgpt)](https://github.com/TheR1D/shell_gpt)** – shell command generation & explanations  
*   **[OpenAI Codex (CLI)](https://openai.com/codex/)** – scriptable repo automation  
*   **[Claude Code (CLI)](https://claude.ai/code)** – interactive terminal coding agent  

### Prompting & Knowledge

*   **[Obsidian](https://obsidian.md/)** / Markdown repo – prompts, runbooks, postmortems  
*   **Reusable prompt patterns** (SRE review, threat modeling, RCA)

Ollama  VS Code + Continue.dev  Claude Code  OpenAI Codex  antigravity  aider  llm  Docker  Git  Obsidian  Prometheus + Grafana  

🔥 Core Takeaways
-----------------

*   AI belongs **in the IDE and the terminal**, not just chat windows  
*   Managers should be **AI-literate, prompt-competent, and code-adjacent**  
*   Terminal AI + local models = **power + privacy**  
*   Tools should reduce toil, not add dashboards
