🧰 IT Engineer / Manager Tool Stack (2026)
==========================================

🤖 AI & LLM Tools (Free / OSS-friendly focus)
---------------------------------------------

### General AI / Thinking

*   **ChatGPT / Claude (free tiers)** – reasoning, docs, debugging, design reviews
    
*   **Local LLMs (Ollama)** – private, offline AI (Llama, Mistral, DeepSeek)
    

### IDE / Vibe Coding

*   **VS Code**
    
*   **Continue.dev (OSS)** – repo-aware AI editing
    
*   **Codeium (free)** – inline completion + chat
    
*   **Cursor-style workflows** (conceptually: multi-file AI edits)
    

### Terminal-First AI (Power-user tier)

*   **antigravity** – natural language → shell, explain/fix commands
    
*   **llm (Simon Willison)** – pipe text/logs/diffs into LLMs
    
*   **aider** – AI pair programming directly in git repos
    
*   **sgpt / shell-gpt** – shell command generation & explanations
    

### Prompting & Knowledge

*   **Obsidian / Markdown repo** – prompts, runbooks, postmortems
    
*   **Reusable prompt patterns** (SRE review, threat modeling, RCA)
    

🧑‍💻 Core Engineering Tools
----------------------------

### Development

*   **Git + GitHub/GitLab**
    
*   **API tools** – Postman / Insomnia
    
*   **Modern browser** – Chrome / Firefox (DevTools)
    

### Containers & Infra

*   **Docker**
    
*   **Kubernetes**
    
*   **Infrastructure as Code** – Terraform, Ansible
    

### CI/CD

*   **GitHub Actions / GitLab CI / Jenkins**
    

📈 Observability, Ops & Reliability
-----------------------------------

*   **Metrics & dashboards** – Prometheus + Grafana
    
*   **Logs & tracing** – ELK stack / cloud logs
    
*   **Incident response** – PagerDuty-style workflows
    
*   **AIOps concepts** – anomaly detection, change correlation
    

🔐 Security & Quality
---------------------

*   **Password manager** – Bitwarden / 1Password
    
*   **Secrets management** – Vault-style tools
    
*   **Static analysis** – SonarQube (OSS)
    
*   **DevSecOps scanning** – Trivy, Snyk-like tools
    
*   **VPN client**
    

📋 Project, Process & Management
--------------------------------

*   **Task tracking** – Jira / Linear / Trello
    
*   **Documentation & KB** – Notion / Confluence / GitHub Wiki
    
*   **Diagramming** – draw.io / Lucidchart
    
*   **ITSM (if applicable)** – Jira Service Management / Freshservice
    

💬 Collaboration & Productivity
-------------------------------

*   **Chat & async** – Slack / Teams
    
*   **Meeting notes / summaries** – built-in AI or OSS tools
    
*   **Email + calendar discipline** (filters > inbox zero)
    

🧠 Foundational CLI & OS Tools
------------------------------

*   **Terminal** – bash/zsh, Windows Terminal, iTerm2
    
*   **Package managers** – brew, apt, winget
    
*   **Search & grep** – ripgrep, fd
    
*   **Editors** – vim/neovim (optional but powerful)
    

⭐ Minimal “Gold Standard” Setup (Free, High-Power)
--------------------------------------------------
Ollama  VS Code + Continue.dev  antigravity  llm  aider  Docker  Git  Obsidian  Prometheus + Grafana   `

🔥 Core Takeaways
-----------------

*   AI belongs **in the IDE and the terminal**, not just chat windows
    
*   Managers should be **AI-literate, prompt-competent, and code-adjacent**
    
*   Terminal AI + local models = **power + privacy**
    
*   Tools should reduce toil, not add dashboards
